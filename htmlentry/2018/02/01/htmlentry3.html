HTML入門[第三回]HTMLを作成してみる

access_time 2018年02月01日

とにかくHTMLを作成してみます

実際に作りながら少しずつ知識を増やすのが良いです。今回は必要最低限のタグを使ってほとんど空のhtmlを作っていきます。

作業フォルダを作成

任意の場所にhtmlファイルを作るフォルダを作成してください。例ではc:\workspace\htmlというフォルダを作成します。自分の好きな名称でよいので作成してよいですが日本語を使ったフォルダは作らないでください。 アルファベット、数字、アンダーバー「_」やハイフン「-」などの一部記号のみを使うのが無難です。

※プログラムが扱うファイル名やディレクトリ名、urlなどは英数字以外を想定していない場合が多くあります。 下記のようなフォルダを作ります。

ファイルを作成

新規作成

今回はTeraPadで作業を進めていきます。Terapadを開き上部のメニューから[ファイル]-[新規作成]をクリックします。

保存

TeraPadのメニューの[ファイル]から[文字/改行コード指定保存]をクリックします。

文字コードに[utf-8]を指定して保存をクリックします。

保存する場所は、先ほど作成したフォルダに作ります。この場合c:\workspace\htmlという場所をしていします。ファイル名はindex.htmlとしてください。

index.htmlはブラウザ等で最初に読み込まれるファイルにつける名前です。たとえばhttp://www.google.co.jpとurlを入力してページを見る場合には実際にはhttp://google.co.jp/index.htmlを読み込もうとします。細かくはサーバーの設定になるのですが特に指定がなければファイル名が省略されている場合にindex.htmlを読み込もうとします。

実際にHTMLを作成していきます

作成したindex.htmlにhtmlを記述していきましょう。今回は必要最低限のタグを記述していきます。

DOCTYPE

まずはhtmlをどのバージョンで書くかを宣言します。この場合はhtml5を書きましょう。とてもシンプルになっているので楽になりました。

<!DOCTYPE html>

htmlタグ

次に全体をhtmlで記述するのでhtmlタグを書き足していきます。またlang=”ja”は日本語で文書を記述しますという宣言に当たります。

<!DOCTYPE html>
<html lang="ja">

</html>

ヘッダとボディタグ

html全体の設定などを記述するhtmlヘッダと実際表示される部分を記述するbodyをくくるタグを記述します。ここからここまではヘッダですといった構造的な意味のみをもちます。このようにhtmlはhtmlタグの中にheadタグ、bodyタグを入れていくという入れ子の構造で記述していきます。

<!DOCTYPE html>
<html lang="ja">
	<head>

	</head>
	<body>

	</body>
</html>
入れ子の構造の親子関係が分かりやすくなるように、子になるタグの記述位置をスペースでずらす（インデントをつけるといいます）のが慣例です。スペースいくつをつけるかというのは特に決まりはありませんがスペース4つ開けるのがよく見られます。

ヘッダの要素

ヘッダタグの中にはhtml文書の設定や概要などを記述します。人間が見る部分ではなく検索エンジンやブラウザにとって必要な情報を記述する部分にあたります。ここではこのhtmlを記述する文字コードが[utf8]であるということと、htmlのタイトルを記述します。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
	</head>
	<body>

	</body>
</html>
htmlファイルやhtmlのheadで文字コードutf8を指定するのは理由があり基本的にはなにも考えずにutf8を利用してかまいません。utf8はアスキーコードと互換性があるためほとんどのコンピュータやプログラムで誤動作しません。その他の文字コードの場合「¥」などの記号が別のコードで使われていたりするため誤動作する場合が多々あります。簡単にいうと文字化けの原因になるので特別な指定がない限りutf8を利用してください。

タイトルタグの表示はブラウザのタイトル欄を見てもらえばわかります。

ボディに要素を追加

ボディタグには試しとして一つの段落をくくる意味でpタグ（ぴーたぐ）で何か文字を記述してみます。ここでは「初めてHTML文書をかいてみます。」という段落を書いてみましょう。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
	</head>
	<body>
		<p>初めてHTML文書をかいてみます。</p>
	</body>
</html>

図のようにボディタグに何かを記述して初めてブラウザに表示されます。

初めてのHTMLを作成しました

今後はこのhtmlをベースとしていろいろなタグを使ってみたりするので現段階でわからない場合は読み返すかわからない箇所を質問してください。以降はよく使うであろうタグの説明や使ってみたところなどをまとめていきます。箇条書きすると下記のことができるようになれば大丈夫です。

  • index.htmlを文字コードutf8で作成
  • DOCTYPE宣言をしてhtml5でhtmlを記述
  • htmlタグ,headタグ,bodyタグをインデントをつけて記述
