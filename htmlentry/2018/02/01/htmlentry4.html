よく使うタグ

きれいに書かれたHTMLでは頻繁に使うタグは決まっています。特殊なデザインのページ以外ではほとんど同じようなタグしか使いません。これらよく使うタグを見ていきましょう。やはりこれも暗記するといったことは必要ありません、ブログや趣味のサイトを作った場合結果的によく使うようになります。

<p>タグ

これは段落を表すタグです。一つの文章の塊で区切りたい場合必要になってきます。この文書もpタグを使って段落を区切っています。また、段落中下記の記述では改行をしていますが見やすいようにしているだけです。エディタで入力されている改行はHTMLでは無視されます。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<p>
				改行コードをいれます。<br>
				しかしこの記述の後には改行を入れません。
				そのためこの文章の前には改行はありません。<br>
				改行タグを入れた位置、入れない位置の違いを確認してください。
			</p>

		</div>
	</body>
</html>

<div>タグ

このタグはdivisionタグ（ディブタグと呼んでます）これは特に意味を持ちませんがデザイン上要素の塊を区切るときに使います。ここからここまではひと塊といったときに使います。

html5ではheaderタグ、articleタグといったdivタグのように要素の塊を区切るときにある程度意味を持たせるタグがあるので塊に意味がある場合にはそれらのタグを使うのがベストです。それより細かく区切るときに使いましょう。（といってもきれいに書くのはとても大変なので気にしないという手段もあります）

上記のように記事見出し一覧みたいな画面で複数の記事を表現するときに区切りとして使ったりする。デザインで透明のままでもよいが背景色を変えたり区切り線をつけたりすることもできる。

<h1>～<h6>

h1～h6タグ。Headingタグの略ですが１～６の順で重要なページ内タイトルです。html構造上とても重要なタグとして位置づけられています。この文章は大きくh1に書かれていることについての文書として認識されます。この情報をGoogleなどの検索エンジンやブラウザや様々なサービスが解析します。

このページでいう次の画像の部分がh2タグh3タグです

下記のように記述します

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<h1>大見出し、文書の内容を象徴するような見出し</h1>

			<p>
				文章の段落１
			</p>

			<h2>中見出し、文章を区切ったそれぞれの内容の見出し</h2>

			<p>
				文章の段落２
			</p>

		</div>
	</body>
</html>

<span>タグ

divタグよりさらに小さい文章の中の一部を装飾したいときに区切るタグ。文章の一部を赤色で表示したい時などに使用します。cssのフォントの知識と前後しますが下記のように記述すると文章の一部の色を変えたりすることができます。divタグでも全く同じことができますが文章の一部の場合はこのspanタグを使うのが一般的です。

[style.css]

p{
    text-indent:1em;
}

.akamoji{
    color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<p>
				段落タグで囲まれた段落のうち一部だけ<span class="akamoji">赤文字</span>
				にしたい場合に利用するタグです。
			</p>
		</div>
	</body>
</html>

<a>タグ

アンカータグ。基本的にはhtmlの一番重要なタグです。このタグで囲まれた文字にはリンクというものが貼られ、クリックをすると指定したurlをブラウザに表示させることができます。

例えばこのリンクをクリックするとGoogleのサイトへ飛びます。これがアンカータグです。

この文章は下記のように記述されています。

<p>
    例えばこの<a href="http://google.com" target="_blank">リンク</a>をクリックすると
    Googleのサイトへ飛びます。これがアンカータグです。
</p>

このタグでは上記のようにhrefというプロパティの設定が必須になっています。hrefというプロパティにクリックしたときに飛ばしたいURLを記述します。

このhrefに前に学んだid属性を指定することもできます。href=”#id-name”とした場合、同一ページにあるid-nameというidがつけられた要素へ画面が移動します。

<img>タグ

イメージタグです。ここには画像を指定することができます。イメージタグにはaタグのように必須のプロパティがあります。イメージタグではsrcというプロパティが画像へのurlとなります。cssと同様にhtmlから見た相対的なurlを指定することが多いです。下記にimgタグを記述する手順を記載します。

イメージ用のフォルダを作ります

以前作成したindex.htmlを保存したフォルダに新規フォルダを作成します。

名前を[img]とします。（一般的に画像を保存するフォルダ名に利用される名前です）

表示させたい画像をimgフォルダにコピペします

任意の画像を今作成したフォルダに貼り付けてください。

imgタグを記述します

下記のscr=の後の”./img/sample.png”の記号「.」は相対パスというurlの指定方法で「同じ階層の」という意味になります。

  • 記号「.」ピリオド一つ:同じ階層を示す
  • 記号「..」ピリオド二つ:一つ上の階層を示す
<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<img src="./img/sample.png" width="500px">
		</div>
	</body>
</html>

リストタグ<ul>タグと<li>タグ

ulとliタグを利用して何かのリストを表す際に記述します。その他一般的にはWebサイトのメニューなどにも利用されます。同じ序列（意味合い）のものを箇条書きするときに使います。ulというのはliの親要素として束ねる意味で記述します<ul>から</ul>までの子要素はひとまとめのリストですという意味のタグです。<li>タグはリストの一つ一つの要素という意味で記述します。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<ul>
			<li>リスト一つ目</li>
			<li>リスト二つ目</li>
			<li>リスト三つ目</li>
		</ul>
	</body>
</html>
ちなみにulタグのほかにolというタグも存在します。ulタグでは順番に意味はな一つ一つのリスト要素は「・」で区切られますがolの場合には昇順で番号がついていきます。今作成したulをolに変えて再読み込みしてみてください。
