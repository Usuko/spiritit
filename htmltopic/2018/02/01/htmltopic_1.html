HTML CSS JavaScriptではこんなことができます

access_time 2018年02月01日

HTML CSS JavaScriptを学ぶと表現する手段が増えます

現在インターネットは高速になり（環境によりますが）スマートフォンなども普及しています。SNSやブログなどを含め一般の個人が自分の趣味や仕事を日本中、世界中に発信しています。それにはHTMLやCSS、JavaScriptといった技術を知っていると一層表現力も上がります。 また、単純にきれいなWebサイトなどは作ること自体も楽しみがあります。実際に優れたデザインと評判のWebサイトなどをみて興味があればHTMLなどを学習することをお勧めします。

※リンク切れする可能性があります

メルセデスベンツ、Eクラスのサイト

スクロールをするとベンツの画像が粒子のように細かく分解され、その粒子が集まって違う画像を表示します。動画なのかJavaScriptでやっているのかはちょっとわかりませんが、いわゆるクールな表現です。サイト全体の色は高級感を出すために黒を基調としていると思われます。

大森センターの環境だと重くて固まってしまうのでリンクを切りました。

東京メトロ

ドラえもんとのコラボレーションなので全体的にはかわいい感じのデザインになっています。色もカラフルですがチカチカはしない良いデザインだと思います。ドラえもんとドラミちゃんがメトロのいろいろな情報を紹介していく感じで楽しげです。

株式会社ロイドコントライブ

店舗内装 企画施工 CGパース製作です。店舗の内装もとても洗練されているように見えます。Webサイトも同様でとてもセンス溢れるWebサイトだと思われます。

伊東養鶏場

このサイトは、私がただ卵が好きなだけなのもありますが卵を大切に作っているんだろうなというのが伝わってくるようなサイトデザインになっています。（実際のところはよくわかりませんが）養鶏所内の動物相関図はちょっとかわいらしいです。

MONO SENDAI

仙台の粋なものとそのお店を掲載するサイトです。現状は見た目ほど多くのものや店がなく同じものが表示されています。ランダムで表示しているのかもっとたくさんあるように見える優れたデザインだと思われます。

松竹芸能

松竹芸能のWebサイトはちょっとセンスがあります。なぜか芸能事務所のWebサイトは忙しいからなのかダサいものが多いです。Webサイトにはお金をかけていないのでしょうかわかりません。芸能人はテレビ以外の仕事結構やってて大変な職業なんだなぁとなぜか関心しました。

これらサイトはプロが作ったサイトと思われます。これらのサイトも結局はHTMLやJavaScriptを利用しています。少し前までは画像や配色だけをデザインするだけのデザイナーもいましたが現在ではWebサイトにかかわる人は必ずHTMLの知識が必要になります。冒頭でも書きましたがHTMLやJavaScriptを知るだけでもこんなことができるんだといった楽しみがあります。またHTML/CSS/JavaScriptは実行が簡単なプログラミング言語でありプログラムを学ぶ入門には最適です。少しでもWebサイトやプログラミングに興味があるなら学び始めることをお勧めします。

HTML/CSS/JavaScriptだけで作れるゲーム

学習用のプログラムとしてとても有名なのですが、HTML/CSS/JavaScriptのみでテトリスが作られて配布されています。HTML/CSS/JavaScriptの重要な技術を結構使っているためとても参考になるプログラムです。一般に公開されているので下記のサイトからダウンロードしてください

ダウンロード

こちらのサイトからプログラムをダウンロードして実行してみます。

開いたサイトの緑のダウンロードボタンをクリックしてください。

ダイアログが出るのでダウンロードのほうを選択してください。

展開

ダウンロードしたフォルダを展開してください。ブラウザやPCによってダウンロードされる先が違いますので注意してください。

実行

展開するといくつかのファイルがあるのがわかりますがindex.htmlというのを実行してください。
（ブラウザで見てください）
