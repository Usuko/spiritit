下記のようにaタグ（アンカータグ）を追加します。アンカータグの子要素としてimgというタグで画像を表示しています。このタグは少し特殊で終了タグがありません。単純にsrcというプロパティに指定した画像を表示します。

<div class="content-body">

	<aside>
		<a href="#"><img src="./img/title.jpg" /></a>
	</aside>

上記での画像を見ると１つの画像に２つの絵が表示されています。これには理由があります。

お手本サイトを見てみるとメニューの部分にマウスカーソルを充てると少し色が変わっているようなアクションになると思います。これはCSSのhover（ホバー）という機能を使って行います。簡単に言うと通常時は左の画像を表示し、マウスカーソルが画像の上に乗った時は右の画像を見せているだけなのです。実際に作ってみましょう。

CSSの編集

現状では見やすいように背景色を設定してしまっているのでいったん背景色をなくします。

aside{
	background-color:blue; //この行を削除
	width:220px;
}

上記bakcground-colorの行を削除しましょう。
そして現状のサイトを確認してみると下記のようになっていると思われます。

CSSを編集して左下のリンクの部分を画像にしていきましょう。

まずリスト全体のスタイル指定

navタグやulタグliタグはデフォルトで設定されている内容があるのでまずそれをリセットします。下記のクラスをstyle.cssに追加します。

.side-menu ul{
	padding:0;
}
.side-menu li{
	list-style-type: none;
}

これはリスト表示の周囲にある余白であるpaddingと各リストの先頭についている「・」の記号を取り除く設定です。

CSSでリンクの画像を設定

まずは画像を表示してみます。上記で追加したliというタグリストタグにmenu01～07を設定したのでひとつづつmenu01から始めます。下記クラスをstyle.cssへ追加します。

.menu01{
	background-image: url(../img/menu01.jpg);
	background-repeat: none;
	height:60px;
	width:200px;
}

CSSでホバー時の画像を設定

上記の画像が表示されたら次は、マウスカーソルが上に乗った時の動作であるホバー(hover）の設定を行います。この方法では１枚の画像の表示位置であるbackground-positionを使って行います。background-positionは値にx軸のサイズである横の開始位置とy軸のサイズである縦の開始位置となります。マウスが乗った時は横に200pxだけずらした画像を表示するよう設定します。

ホバーの設定をするときにはセレクタの後に「:hover」と指定します。

.menu01:hover{
	background-image: url(../img/menu01.jpg);
	background-repeat: none;
	background-position: 200px 0px;
	height:60px;
	width:200px;
}

現在の表示を見てみましょう。画像が表示されマウスカーソルが乗ると画像が変わるようになっているはずです。

この文字を消すにはcssに下記のコードを追加しましょう。これは現在重なっている要素の順番を文字列であるspanタグを一番下にするという設定です。

.side-menu span{
	position:relative;
	z-index:-1;
}

すべて同じように07まで行ってください。 ただし07のメニューだけwidthが44pxですので注意が必要です。

お問い合わせを作成

メニュー部分の最後であるお問い合わせを作成していきます。この部分は画像とテキストの組み合わせで表示されています。 まずは下記のお問い合わせ画像をダウンロードしてimgフォルダに入れてください。

htmlの編集

htmlを編集してaddressを表示するためのdivタグを挿入しましょう。そしてその中に住所を記述する専用のタグであるaddressというタグを追加します。下記のように先に追加したnavタグの下に追加します。asideは現在下記のようなタグになっています。

	<nav class="side-menu">

CSSの編集

次にCSSを編集しましょう。addressの背景画像を先ほどの画像にします。bodyやheaderなどと同様classではなくタグを指定するのでCSSで指定するときは[.]記号を先頭につける必要はありません。下記の内容を設定しています。

  • 画像の高さは175px
  • 画像の横幅は200px
  • 住所のフォントスタイルはnormal（addressタグのデフォルトは斜体になっています）
  • 住所の１行ごとの高さは19.5px
  • 住所の書き始めの高さは上から105px
  • 住所の書き始めの左からの幅は20px
address{
	background-image: url(../img/address.jpg);
	height:175px;
	width:200px;
	font-style:normal;
	line-height:19.5px;
	padding-top:105px;
	padding-left:20px;
}
