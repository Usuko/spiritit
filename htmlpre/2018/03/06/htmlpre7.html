HTML初級[第七回]-メイン部分を作成(1)

access_time 2018年03月06日

メイン部分の作成

メイン部分は上から順に作成していきましょう。

コンテンツヘッダ

デザイナさんが書いたメインの絵をコンテンツ部分の先頭に表示されています。これを設定していきます。

画像のダウンロード

最後にメイン部分を作成していきます。これは単純にコンテンツトップ画像を表示しています。下記の画像を今までのやり方と同じようにimgフォルダにダウンロードします。

imgフォルダの内容は現在このようになっているはずです。

今ダウンロードした画像をメイン部分の一番上に設置します。

HTMLの追加

HTMLの下記の部分を編集します。

<main>
	メイン
</main>

これを下記のように編集します。

<main>
	<img src="./img/main.jpg" />
</main>

私たちの願い

私たちの願いの部分をひとつづつ作成していきます。サンプルのサイトではこの「私たちの願い」の文字はh2タグを指定されています。h2タグはこのページの中でも重要な文字につけられるタグです。h1、h2といったタグはコンピュータがこのページの内容を判断するタグとして使います。

ここでもそれを使いますが表示は画像なので、今までと同様下記の画像をまずダウンロードしてください。

小見出し（私たちの願い）の追加

htmlのimgタグの下に下記のタグを追加します。下記のように追加されればOKです。

<main>
	<img src="./img/main.jpg" />

	<!-- ここから追加 -->
	<div class="our_wish">
		<h2>私たちの願い</h2>
	</div>
	<!-- ここまで追加 -->

</main>

そして現在、初めにわかりやすいようにしたため背景が緑色になっています。まずはこれは元に戻します。style.cssのmainを設定した箇所のbackground-colorの行を削除します。

main{
	background-color:green;  //この行を消します。
	width:700px;
}

そして、私たちの願いを表示する部分のクラスを追記します。またこの場合お問い合わせのように画像の上に「私たちの願い」という文字がひょうじされてしまうのでh2タグについて表示を背後にするx-indexで設定を行います。

.our_wish{
	background-image: url(../img/h2_1.jpg);
	background-repeat: no-repeat;
	background-color: white;
}

.our_wish h2{
	position: relative;
	z-index: -1;
}

リストの追加

小見出しの下の５つの願いのリストを追加します。このリストの先頭のチェックボックスのような部分は画像なのでこれを使います。下の画像を例によってimgフォルダにダウンロードしてください。

そしてHTMLを修正します。私たちの願いの下にリストを表すulタグliタグを追記します。

<main>
	<img src="./img/main.jpg" />

	<div class="our_wish">
		<h2>私たちの願い</h2>

		<!-- ここから追加 -->
		<ul>
			<li>社会参加したいがどうしたらいいか、わからない</li>
			<li>就職への自信が持てない</li>
			<li>コミュニケーションが苦手で、人前に出るのが不安</li>
			<li>生活のリズムをつけて社会参加したい</li>
			<li>身体や心の悩みを抱えているが、仕事がしたい</li>
		</ul>
		<!-- ここまで追加 -->

	</div>

</main>
ulタグliタグはセットです。リストアイテムを示すliタグをまとめるのがulです。olというタグもありますがそれは別途調べてください

今現在の表示はこのようになっていると思われます。

参考のサイトは下記のように少し細かい設定が必要です。

これを表現したCSSは下記です、すでに記述済みである.our_wishとour_wish h2の記述をいったん消して、下記のコピーを貼り付けてみてください。確認等は行を削ったりしてみて各自行うのが分かりやすいです。

.our_wish{
	background-image: url(../img/h2_1.jpg);
	background-repeat: no-repeat;
	background-color: white;
	padding-top: 20px;
	padding-left: 30px;
}

.our_wish h2{
	position: relative;
	z-index: -1;
}

.our_wish ul{
	list-style-type: none;
	padding: 0;
}
.our_wish li{
	background-image: url(../img/box.jpg);
	background-repeat: no-repeat;
	background-position: 0 9px;
	line-height: 100%;
	font-size: 120%;
	font-weight: bold;
	color: #2087B9;
	padding: 10px 0 10px 30px;
}

このような・・から始まる文言の追加

参考のサイトにある「このような・・・」という文字をすべてコピーしてhtmlに追記します。このように文字を追加する場合pタグ（パラグラフ）というものでくくるのが一般的です。

HTMLの修正

下記のようにulタグの下にpタグを追加しその要素に文言を貼り付けてみましょう。

<div class="our_wish">
	<h2>私たちの願い</h2>

	<ul>
		<li>社会参加したいがどうしたらいいか、わからない</li>
		<li>就職への自信が持てない</li>
		<li>コミュニケーションが苦手で、人前に出るのが不安</li>
		<li>生活のリズムをつけて社会参加したい</li>
		<li>身体や心の悩みを抱えているが、仕事がしたい</li>
	</ul>

	<!-- ここから追加 -->
	<p>
		このような悩みを抱えている方を、就職や社会参加に向けて支援するセンターです。
		皆さんが「自信」「自己肯定感」を持って元気になり、社会参加の希望がかなえられる
		──それが、Spirit大森センターの願いです！

		ぜひ一度、見学にいらしてください。

		【皆さまの声】は こちら ──Spirit大森センターを卒業された先輩やご家族の声をお聞きください！
	</p>
	<!-- ここまで追加 -->

</div>

この段階でブラウザで見てみましょう。

改行がないことになっています HTMLはこのようにHTMLファイルで改行を入れてもブラウザでは改行してくれません。そのためにHTMLでは文章を改行する際に<br>というタグで指定します。先ほどのHTMLを再編集して改行してみましょう。

<div class="our_wish">
	<h2>私たちの願い</h2>

	<ul>
		<li>社会参加したいがどうしたらいいか、わからない</li>
		<li>就職への自信が持てない</li>
		<li>コミュニケーションが苦手で、人前に出るのが不安</li>
		<li>生活のリズムをつけて社会参加したい</li>
		<li>身体や心の悩みを抱えているが、仕事がしたい</li>
	</ul>

	<!-- brタグを追加 -->
	<p>
		このような悩みを抱えている方を、就職や社会参加に向けて支援するセンターです。<br>
		皆さんが「自信」「自己肯定感」を持って元気になり、社会参加の希望がかなえられる<br>
		──それが、Spirit大森センターの願いです！</br>
		<br>
		ぜひ一度、見学にいらしてください。</br>
		<br>
		【皆さまの声】は こちら ──Spirit大森センターを卒業された先輩やご家族の声をお聞きください！<br>
	</p>
</div>

そしてCSSですが行の高さがサンプルと異なります。サンプルでは行の高さは180%となっているのでCSSでpタグの内容分の行の高さを180%に設定します。style.cssへ下記設定を追記します。

CSSの修正

p{
	line-height: 180%;
}

文末の微修正

皆様の声はという部分は少し太字で大きい文字になっているのでこれも同じように設定してみましょう。これは太字で且つfont-sizeがmediumです。まず文字を飾る場合に使用するspanというタグでhtmlを修正します。

またそのすぐ後の「こちら」という文字に関してはリンクが張られています。リンクが張られているとは今まで作成してきたメニューのようにクリックすると別のページに行く仕組みです。こちらは以前もつかったaタグ（アンカータグ）を使います。font-sizeはmediumでcolorは#EE6C02です。一緒に修正してみましょう。

<p>
	このような悩みを抱えている方を、就職や社会参加に向けて支援するセンターです。<br>
	皆さんが「自信」「自己肯定感」を持って元気になり、社会参加の希望がかなえられる<br>
	──それが、Spirit大森センターの願いです！</br>
	<br>
	ぜひ一度、見学にいらしてください。</br>
	<br>
	<!-- spanやaタグを追加 -->
	<span class="voices">【皆さまの声】は</span><a class="here" href="#">こちら</a> ──Spirit大森センターを卒業された先輩やご家族の声をお聞きください！<br>
</p>

そしてCSSには以下の記述を追記します。style.cssの最後に貼り付けます。

.voices{
	font-weight: bold;
	font-size: medium;
}

.here{
	font-size: medium;
	color: #EE6C02;
}

微調整

私たちの願いの一番下に少し余白があるのでこれも直しておきます。

our_wishにpadding-bottomというプロパティを追記します。下の余白を設定するときのプロパティです。

.our_wish{
	background-image: url(../img/h2_1.jpg);
	background-repeat: no-repeat;
	background-color: white;
	padding-top: 20px;
	padding-left: 30px;
	padding-bottom: 20px;
}
