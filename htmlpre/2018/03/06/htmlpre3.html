参考ではc:\workspace\htmlというフォルダに作成します。どこでも構いません。 下記のような構成になります。

index.htmlを開いて作業を続けます。

レイアウトを構成を記述

まずはレイアウトを作っていきます。Webサイトのメインで表示される部分は基本的にbodyタグの中身なので。まずはbodyタグの中にレイアウトを記述していきます。

bodyタグの中といった場合、<body>から</body>に挟まれた間のことを指します。

前回のまとめですと大きなレイアウトは下記の４つのパーツから構成されています。

  • ヘッダ
  • メニュー
  • メイン
  • フッター

大きなレイアウトを構成する場合基本的にはdivというタグを使います。Divisionの略で分割といった意味です。意味のとおりそのdivタグで意味を分割します。手始めにこのdivタグをレイアウト分４つ作成します。先ほどのindex.htmlを編集し下記のように変更してください。

<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<title>サンプル</title>
</head>
<body>

	<div>
		ヘッダ
	</div>

	<div>
		メニュー
	</div>

	<div>
		メイン
	</div>

	<div>
		フッタ
	</div>

</body>
</html>

編集が終わったらindex.htmlをダブルクリックして開いてみてましょう。 ヘッダ、メニューなどといった文字は表示されますが目的のようなレイアウトにはなっていません。

気づいてる人もいると思いますが、まだメニューとメインの部分は横に並べるなどといった表示に関する内容をどこにも書いていません。 頭の中には上の図のようなレイアウトにしたいのですがそれをコンピュータに教えなくてはいけません。それがCSSです。

html5の書き方をする

と、今までであれば基本的にレイアウトはdivというタグで区切っていました。ですが現在のほとんどのWebサイトはこれと同じような構成でつくられています。見出しなどをコンピュータにわかりやすくするためにタグも増えました。

折角なので存在するタグは利用してみましょう。上記のコードを再度編集して下記のコードのしてください。

<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<link rel="stylesheet" href="./css/style.css">
	<title>サンプル</title>
</head>
<body>

	<header>
		ヘッダ
	</header>

	<aside>
		メニュー
	</aside>

	<main>
		メイン
	</main>

	<footer>
		フッタ
	</footer>

</body>
</html>

divタグをそれぞれheader,aside,main,footerタグというものに変えました。

headerはそのままヘッダーですという意味です。
同じようにfooterはフッターです。
mainはブログなどで言ったらそのWebサイトで伝えたい内容を表示する部分です。
asideは基本的にはメニューであったり最近の記事へのリンク、広告などの他のコンテンツやサイトへのリンクをまとめることが多いです。

ダブルクリックして開いても表示に変化はありません。なぜこのようなことをするのかというとコンピュータが読みやすいようにしています。このタグを使うことでコンピュータはmainの部分がこのサイトの構成を把握できます。そのためコンピュータが利用しやすいサイトになります。

分かりやすいようにCSSを適用して背景色を変える

今のままデザインを変更していっても少しわかりにくいです。いったんこの部分ごとに背景色をつけて今どのようにレイアウトされているかを確認していきましょう。

先ほど作成したstyle.cssに下記記述をコピーして貼り付けてください。

header{
	background-color:red;
}
aside{
	background-color:blue;
}
main{
	background-color:green;
}
footer{
	background-color:yellow;
}

index.htmlを再度読み込むと下記のように色が分かれて表示されます。これで今どのようにレイアウトされているかがわかります。現状では単純に四角いエリアが四段重なっているだけですね。

それではひとつづサイトを目標に近づけていきましょう。

横並びのレイアウト

次に、レイアウトでコンテンツを横並びにする方法です。 今記述してきたHTMLではメニュー部分のasideタグとmainタグは縦に並んでいます。これを図のように横に並べていきます。

htmlの修正

横に並べるためには横に並べたい要素を一つのdivに入れる必要があります。 のでhtmlは下記のようにasideとmainタグをひとつのdivタグの中に入れてください。

<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<link rel="stylesheet" href="./css/style.css">
	<title>サンプル</title>
</head>
<body>

	<header>
		ヘッダ
	</header>

	<div class="content-body">

		<!-- このようにasideとmainタグを含むようにdivを追加 -->
		<aside>
			メニュー
		</aside>

		<main>
			メイン
		</main>

	</div>

	<footer>
		フッタ
	</footer>

</body>
</html>

上記のようにcontent-bodyというクラスのdivタグを追加します。

divタグの挿入

divタグを追加してみましょう。
asideとmainを囲みたいので、まずはheaderの終了タグとasideの開始タグの間にdivの開始タグを挿入します。classをcontent-bodyとして指定しておきます。

divの終了タグを挿入

開始したタグは終了しましょう。
mainの終了タグとfooterの開始タグの間にdivの終了タグを挿入します。

見やすいように整形

htmlやcssなどのコードは見やすいようにしておきましょう。
たった一人で作る場合でも１日２日たった後見るとその時のことは思い出せない場合もあります。２日後の自分は他人とおもってわかりやすいようにするのが良いです。

インデントというものをつけます。同じ要素の位置を揃えるのが目的です。
asideとmainの要素は追加したdivタグの子要素になります。子要素はインデントという鋳物をつけて表すのが通例です。

asideとmainの要素をドラッグして反転します。

反転した状態でキーボードの「Tab」キーを一回入力します。

これがインデントが付いた状態でわかりやすいとされています。

Tabキーを押しすぎてしまった場合、「Shiftキーを押しながらTab」を一回押してください。インデント一つ分元に戻ります。

cssへの追記

htmlでasideとmainタグを含む新しいdivを追加しました。このclassであるcontent-bodyというクラスを追加することでdivの中の要素の表示方法を設定できるようになります。

それではcssへ下記のクラスと追加していきます。

.content-body{
	display: flex;
	flex-direction: row;
}
このdisplay:flexというのはcontent-bodyを指定した要素の子要素はcontent-itemという要素として取り扱われます。 このcontent-itemはflex-direction: rowというプロパティで横に並ぶようになります。 その他flexというのはflexboxと呼ばれる技術でcss3くらいから普通になりましたがとても便利です。この横並びもflexbox以前はとても複雑に行っていました。

そして、上記.content-bodyの部分をコピーしてファイルの最後に貼り付けます。

必ずしも最後である必要はありません。セレクタ{プロパティ:値}の塊の間であれば構いません。他のクラスの内容の途中である場合にはエラーとなりエラーとなった個所以降のCSSが適用されなくなってしまいます。


確認

表示を確認してみましょう。横並びになっているはずです。
