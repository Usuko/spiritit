参考ではc:\workspace\htmlというフォルダに作成します。どこでも構いません。 下記のような構成になります。

index.htmlを開いて作業を続けます。

レイアウトを構成を記述

まずはレイアウトを作っていきます。Webサイトのメインで表示される部分は基本的にbodyタグの中身なので。まずはbodyタグの中にレイアウトを記述していきます。

bodyタグの中といった場合、<body>から</body>に挟まれた間のことを指します。

前回のまとめですと大きなレイアウトは下記の４つのパーツから構成されています。

  • ヘッダ
  • メニュー
  • メイン
  • フッター

大きなレイアウトを構成する場合基本的にはdivというタグを使います。Divisionの略で分割といった意味です。意味のとおりそのdivタグで意味を分割します。手始めにこのdivタグをレイアウト分４つ作成します。先ほどのindex.htmlを編集し下記のように変更してください。

<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<title>サンプル</title>
</head>
<body>

	<div>
		ヘッダ
	</div>

	<div>
		メニュー
	</div>

	<div>
		メイン
	</div>

	<div>
		フッタ
	</div>

</body>
</html>

編集が終わったらindex.htmlをダブルクリックして開いてみてましょう。 ヘッダ、メニューなどといった文字は表示されますが目的のようなレイアウトにはなっていません。

気づいてる人もいると思いますが、まだメニューとメインの部分は横に並べるなどといった表示に関する内容をどこにも書いていません。 頭の中には上の図のようなレイアウトにしたいのですがそれをコンピュータに教えなくてはいけません。それがCSSです。

html5の書き方をする

と、今までであれば基本的にレイアウトはdivというタグで区切っていました。ですが現在のほとんどのWebサイトはこれと同じような構成でつくられています。見出しなどをコンピュータにわかりやすくするためにタグも増えました。

折角なので存在するタグは利用してみましょう。上記のコードを再度編集して下記のコードのしてください。

<!DOCTYPE html>
<html lang="ja">
<head>
	<meta charset="utf-8">
	<link rel="stylesheet" href="./css/style.css">
	<title>サンプル</title>
</head>
<body>

	<header>
		ヘッダ
	</header>

	<aside>
		メニュー
	</aside>

	<main>
		メイン
	</main>

	<footer>
		フッタ
	</footer>

</body>
</html>

ダブルクリックして開いても表示に変化はありません。なぜこのようなことをするのかというとコンピュータが読みやすいようにしています。このタグを使うことでコンピュータはmainの部分がこのサイトの構成を把握できます。そのためコンピュータが利用しやすいサイトになります。

分かりやすいようにCSSを適用して背景色を変える

今のままデザインを変更していっても少しわかりにくいです。いったんこの部分ごとに背景色をつけて今どのようにレイアウトされているかを確認していきましょう。

先ほど作成したstyle.cssに下記記述をコピーして貼り付けてください。

header{
	background-color:red;
}
aside{
	background-color:blue;
}
main{
	background-color:green;
}
footer{
	background-color:yellow;
}

index.htmlを再度読み込むと下記のように色が分かれて表示されます。これで今どのようにレイアウトされているかがわかります。現状では単純に四角いエリアが四段重なっているだけですね。

それではひとつづサイトを目標に近づけていきましょう。

コンテンツの幅を設定する

いったんお手本であるSpiritのサイトを見てみましょう。ウインドウのサイズの幅を伸び縮みさせているとわかるのですがこのサイトに限らずほとんどのサイトはコンテンツの幅が決まっていて画面の中央に表示されています。

余った部分が余白として伸び縮して中のコンテンツの幅は固定されています。　　

このコンテンツの幅を固定して中央に表示するにはまずサイト全体をdivタグの中に入れる必要があります。 ここで、サイト全体といった場合bodyタグの下に当たります。ブラウザで表示されている部分はbodyタグの中が基本です。

まずコンテンツ全体というものを把握します。下記の図のようにbodyの中の要素をさしてサイト全体といっています。
　

ではサイト全体をdivタグの中に入れるという作業を行います。

divタグを挿入してサイト全体を区切る

タグの挿入

まずbodyタグとサイト全体の初めのタグであるheaderタグの間にdivタグ（クラスにcontentを指定）を挿入します。
　

終了タグの挿入

始めたタグは閉じます。サイト全体であったfooterの終了タグとbodyの終了タグの間にdivの終了タグを挿入します。

CSSに上記で指定したクラスを追加する

クラスを追加してみます。ついでにコンテンツの幅を設定しコンテンツを中央に配置してみます。

cssを開き先ほどhtmlで指定した.contentというクラスを追記します。 cssを追記するといった場合下記のように、セレクタからプロパティをまとめる記号の{}で一つのまとまりとして追加します。この設定の記述がひとまとめなので重なったりする記述はありません。 　

このクラスの定義の{　}の間に幅などを指定します。今回指定するのは幅と余った幅は余白を表示するという設定です。
　

marginとはその要素の周囲の余白の設定を指します。autoは余白がある場合中身の要素が中央に表示されるよう自動で計算してくれるプロパティ値です
