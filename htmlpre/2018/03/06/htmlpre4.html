HTML初級[第四回]-全体のレイアウト作成(2)

access_time 2018年03月06日

コンテンツの幅を設定する

いったんお手本であるSpiritのサイトを見てみましょう。ウインドウのサイズの幅を伸び縮みさせているとわかるのですがこのサイトに限らずほとんどのサイトはコンテンツの幅が決まっていて画面の中央に表示されています。

そして余った部分が余白として伸び縮して中のコンテンツの幅は固定されています。
　

このコンテンツの幅を固定して中央に表示するにはまずサイト全体をdivタグの中に入れる必要があります。 ここで、サイト全体といった場合bodyタグの下に当たります。ブラウザで表示されている部分はbodyタグの中が基本です。

今作るサイトの場合、幅はサイト全体ではなくasideとmainの幅だけでウインドウサイズとの差分は余白として表示します。

aside,mainの幅を指定する

現在あるサイトから幅を調べる方法は今回割愛します。
結論から言うとasideは220px,mainは700pxのデザインです。

それでは先ほどと同様にcssを編集します。

header{
	background-color:red;
}
aside{
	background-color:blue;
	width:220px;
}
main{
	background-color:green;
	width:700px;
}
footer{
	background-color:yellow;
}

.content-body{
	display: flex;
	flex-direction: row;
}

asideとmainにwidthというプロパティを追記しました。

結果を見てみましょう。 幅はよいですが、現在何も指定していないため左側に寄っています。

中央表示

この方法は本当にたくさんありますが現在flexboxという強力なcssが一般的になっているのでそれを利用します。 それ以外の方法は少し記述する内容が多いですがこの場合１行追記するだけです。

先ほど触れたflexboxには中身の要素を中央よせするというプロパティを持っています。justify-contentというプロパティです。下記のようにcontent-bodyにプロパティを追加してください。

.content-body{
	display: flex;
	flex-direction: row;
	justify-content: center;  //←この行を追加
}

上記のようになっていればひとまず大きなレイアウトに関して完了です。 次からはコンテンツを人エリア毎追加していきましょう。

背景色とフォント

Spiritのサイトは背景色が決まっています。全体の背景色とフォントをまず合わせておきましょう。CSSファイルを修正して下記のようにしてください。このbodyを設定したクラスをcssファイルの先頭に貼り付けてください。

body {
	font-size: small;
	font-family: verdana,"ヒラギノ角ゴ Pro W3","Hiragino Kaku Gothic Pro",Osaka,"ＭＳ Ｐゴシック","MS PGothic",Sans-Serif;
	color: #535353;
	background-color: #F8F3ED;
}

サイト全体のフォントはsmallという大きさで、フォントは上にあげたフォントからこの文章を読むパソコンにあるものを使っていきます。背景色は#F8F3EDという色を使うという意味です。
