HTML初級[第五回]-ヘッダを作成

access_time 2018年03月06日

ヘッダのコンテンツを確認

ヘッダで表示しているのは事業番号などテキストのみです。

ですが、ヘッダの一番上を見てみてください。２本のラインと少し立体感を出すための影があります。これは実は同じ画像を横に並べたデザインになっています。css3であれば画像を使わずに同じことができますが下記の画像を使って見ましょう。

まずはこの下の画像をダウンロードしてcssと同じようにimgというフォルダを作成して保存します。

ダウンロードの仕方

ダウンロードするにはまず上の画像を右クリックします

表示されたメニューから「名前を指定して保存」を選びます

表示された画面（ダイアログといいます）でcssのあるフォルダまで移動します。
そこで何もないところで右クリックをします。

表示されたメニューから[新規作成]-[フォルダ]を選びます

フォルダ名にimgをつけて開きます。

保存ボタンを押しますファイル名はおそらくback.jpgとしてください。

背景イメージに今のファイルを指定

今保存したファイルをheaderタグの背景として設定します。 cssを編集して下記のようになるよう修正してください。

バックグラウンドイメージを指定

header{
	background-image: url(../img/back.jpg);
}

headerの色を消してbackground-imageというプロパティを設定してください。これは背景に画像を使いますというプロパティです。url(・・・)という部分はその画像の場所を示しています。上記の”../img/back.jpg”の[..]という記号はこのcssファイルから見て「一つ上のフォルダの」という意味です。

「一つ上のフォルダのimgというフォルダにあるback.jpgです」という意味になります。これは相対パスという指定の方法ですが今は覚えなくてもよいです。htmlを学んでいればいろいろなところに出てきて必然的に覚えます。

同じフォルダを指す場合[..]の記号が[.]と一つになります。

index.htmlをブラウザで確認すると下記のようになっているはずです。

高さを指定

少し高さが低いのでお手本の高さに合わせます。この高さはお手本では60pxになっています。CSSを編集して高さを60pxにしてみましょう。高さを指定するプロパティはheightというプロパティです。

header{
	background-image: url(../img/back.jpg);
	height: 60px;
}

下記のように２列表されてしまったはずです。
background-imageは特に指定がない場合敷き詰めるように表示してくれます。

イメージの並べ方を指定

繰り返し横にだけ並べたいのでCSSを編集しましょう。この繰り返しを設定するプロパティはbackground-repeatというプロパティです。

header{
	background-image: url(../img/back.jpg);
	background-repeat: repeat-x;
	height: 60px;
}

上記のように修正してください。これも順番は何でもよいのですが見やすいようにbackgroundのものはまとめて記述してあります。
表示結果が下記のようになるはずです。

背景にイメージを利用するやり方はいったん終了です。

CSSノーマライズ

ここでガラッと話を変えて、ノーマライズというのを行います。詳細はこちらを参考にしてください。

簡単に言ってしまうと、HTML文書を見るブラウザはたくさんありますWindowsとMacを使うかで標準のブラウザも違うしスマホの機種などでも変わってきます。この差をいったんすべてなくすために必要な作業です。

今作ったページをChromeでよく見るとサイト全体の表示に少し隙間があります。
これは実は隙間がなく見えるブラウザもあるのです。これはノーマライズまたはリセットされていないデザインで起こります。

ノーマライズをする

style.cssのあるフォルダにnormalize.cssというフォルダを作成してください。style.cssと同様にuft-8という文字コードで保存してください。

そしてこちら のサイトへいき、内容をすべてコピーします。

先ほど作成したnormalize.cssにその内容を貼り付けて保存します。

そしてhtmlを修正してそのnormalize.cssへリンクを張ります。下記のようになるようheaderにタグを追加してください。

<html lang="ja">
<head>
	<meta charset="utf-8">
	<link rel="stylesheet" href="./css/normalize.css">
	<link rel="stylesheet" href="./css/style.css">
	<title>サンプル</title>
</head>

このnormalize.cssとstyle.css**の順番は厳守してください。normalize.cssはどのcssより先に読み込まれなければその内容が変更されません。

先ほどの少しあった隙間がなくなっていればノーマライズは完了です。

内容を記述

現在ヘッダという文字を記述してある箇所を、掲載してある事業所番号をコピーして貼り付けてみましょう。

<body>

	<header>
		ヘッダ
	</header>

を修正して下記になるよう修正

<body>

	<header>
		【就労移行支援事業】東京都許可（指定事業所番号1311102238）　一般社団法人Spirit運営
	</header>

そうです、現在は表示位置を何も指定していないため左上に詰めて表示されています。

他のコンテンツに合わせて右寄せ

本来はメインコンテンツの右側にそろえて表示されていて、画面上部から26pxの隙間があり、下のメインコンテンツとの間は15pxあります。 また文字の大きさはサイトで使っている文字より少し小さい85%の大きさになっています。これを指定していきましょう。

htmlの修正

ヘッダーはバックグラウンドイメージがあるので幅は指定できません。そのためasideとmainの時にしたようにheaderにも内側にdivを設けて設定しましょう

<body>

	<header>
		<div class="container right">
			<span class="company-no">【就労移行支援事業】東京都許可（指定事業所番号1311102238）　一般社団法人Spirit運営</span>
		</div>
	</header>

上記のように修正してください。クラスをcontainerとright二つ指定できます。containerはfooterでも同じような目的で使うように右寄せするためのrightと分けて作ります。

cssの修正

まずヘッダの修正、先ほどしたcontent-bodyの時と同様flexboxとして指定し内容を中央よせで表示するようにします。また事業番号の記述の上下の隙間であるpaddingというプロパティを設定します。

header{
	background-image: url(../img/back.jpg);
	background-repeat: repeat-x;
	height: 60px;
	display: flex;
	justify-content: center;
	padding-top: 26px;
	padding-bottom: 15px;
}

そして、下記の３つのクラスを追加します

.container{
	width: 920px;
}

.right{
	text-align: right;
}

.company-no{
	font-size: 85%;
}

containerは下に表示されているasideとmainの幅を足したものです。下の表示と幅を合わせるときに使う手法です。

rightは文字を右寄せにするクラスです。

company-noはここでしか使わなそうなのでこちらもクラスを分けました。　

ところが、index.htmlを確認してみるとなんだかヘッダとメインの間が空きすぎてしまいます。

これはpaddingの計算の仕方がブラウザによって異なることで起こります。headerに指定した60pxという高さと要素の隙間を指定した26pxおよび15pxというサイズをすべて足してしまう計算方法をとる場合もあるのです。これを防ぐためにstyle.cssへ追記してください。場所は一番先頭が良いです。

*{
	box-sizing: border-box;
}

簡単な表現でいうことがむつかしい内容ですが、CSSで高さが指定されている場合、要素の高さを計算する際に内容のpaddingやmarginは計算に含まないという内容です。こちらで説明していますので参考にしてください。

下記のよう想定していない隙間がなくなっていればOKです。

これでいったんヘッダに関しては終了です。
