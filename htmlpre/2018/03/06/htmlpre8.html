メイン、メニュー部分の作成

こちらはメニューの時とほぼ同様の作業を行います。navタグとliタグでリンクのリストを作ります。

下記画像６つを例によってimgフォルダへダウンロードしてください。

HTMLの追記

HTMLはmainの終了タグと前回まで作成したdivの間にnavなどのタグを追加して下記のようになります。

		</div>

		<!-- ここから追加 -->
		<nav class="main-menu">
			<ul>
				<li class="menu1"><a href="#"></a></li>
				<li class="menu2"><a href="#"></a></li>
				<li class="menu3"><a href="#"></a></li>
				<li class="menu4"><a href="#"></a></li>
				<li class="menu5"><a href="#"></a></li>
				<li class="menu6"><a href="#"></a></li>
			</ul>
		</nav>
		<!-- ここまで追加 -->

</main>

現在のままですと下記図のようにならびが縦になってしまいます。全体レイアウトでつかった方法を使い横並びにします。 flexboxを使います。style.cssに下記コードを追加します。

.main-menu ul{
	display:flex;
	flex-direction:row;
	padding:0;
}
.main-menu li{
	list-style-type: none;
	z-index:0;
}

これで横になるはずです。そして背景画像をasideのメニューで行ったようにbakcground-imageを使って設定します。上記の記述の後にさらに下記CSSを追加します。画像のひとつづつのサイズはheightが100pxでwidthが120pxです。menu1のサービス内容だけを追加して見てみましょう。

今回も何もしない場合の設定とhover(マウスカーソルを画像の上に乗せた場合）時の設定を記述します。

.main-menu .menu1{
	background-image: url(../img/menu1.jpg);
	background-repeat: none;
	width:120px;
	height:100px;
	margin: 0px 2px;
}
.main-menu .menu1:hover{
	background-image: url(../img/menu1.jpg);
	background-repeat: none;
	background-position: 0px 100px;
	width:120px;
	height:100px;
	margin: 0px 2px;
}
