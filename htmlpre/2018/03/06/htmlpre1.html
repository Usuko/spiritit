HTML初級[第一回]-準備(textエディタ編)

access_time 2018年03月06日

Webサイトを作る場合には下記のような手順を取ります。

  • 発信したい内容を決める（おいしいお店を他人にも見せたいなど）
  • 表示する内容を洗い出す（お店の画像、住所、連絡先、おすすめメニューなど）
  • 洗い出した内容などを基にデザインする
  • デザイン通りhtmlで記述する
  • 公開する

ここではデザイン通りhtmlで記述することのみを取りまとめ学んでいこうと思います。 ただ、htmlを学ぶ上でデザインはとても重要なのですでにあるWebサイトとほぼ同じものを作っていきながら学んでいきましょう。

目標設定

Spiritのトップページをテキストエディタで作成します。下記のページです。

必要なもの

今回学ぶ上で必要なのはテキストディタのみです。 巷には様々なエディタがあって便利な機能などもたくさんあります。ワープロソフトのようにある程度文字を入力すると予測してコードを書いてくれるものやhtmlのコードを色分けしてわかりやすくしたり様々です。

が

テキストエディタ、html初めてという場合はTeraPadを利用してください。 テラパッドの用意は別途こちら参照してください。

なぜかというと・・・

最初から予測変換などフルの補助機能を使ってしまうとそのソフトには慣れて早くなるかもしれません。 ところがツールが裏で何をしてくれているかをわからない場合あとあと少し困ったことになります。 htmlを書くPCなどの環境が変わったとたん何もできなくなってしまったり新しいツールになれる時間が必要になってしまいます。

htmlを学ぶことが重要ではない場合にはブログなどのサービスを利用することのほうが早いです。現在のブログなどのサービスははほとんどHTMLがわからなくても簡単に記事を書くことができます。

説明に使う用語

説明する際にHTMLでの用語を時々使いますが下記の用語は入門などで説明済みなので意味などは割愛します。

説明中出てきそうな用語はこちらに簡単な説明を書いておきます。わからないときはちらちら見ながら説明を読んでみてください。

HTML

下記のような記号だらけのページですが説明する際に下記の用語を使います。詳細については別途HTMLの入門などを見てください。

タグ

記号の大なり小なりで囲まれた文字をタグと呼んでいます。このtagnameの部分に何を書くかで意味が変わってきます。下記のようにこの何を書くかの部分をタグ名と呼んでいます。

開始タグ　終了タグ

タグには開始タグと終了タグがあり、終了タグは開始タグの最初にバックスラッシュ（/）の記号が付いたものになります。 基本的に、すべてのHTMLの内容は開始タグと終了タグに挟まれて書かれることでコンピュータが意味を理解できます。

下記では「文章あああああ」という内容をスパンタグで囲ってあり、コンピュータは「文章あああああ」をspanであると解釈してディスプレイに表示します。

要素　子要素

タグで囲まれた部分を要素（時にはまとめてタグといいます）と呼びます。また、要素の中にもタグは記述できます。その際中のタグは子要素といわれます。

CSS

セレクタ

CSSではHTMLを修飾する際に修飾する先を指定します。その指定する記号の部分をセレクタと呼びます。 セレクタの種類は様々ありますが別途入門などを見てください。

プロパティ

セレクタで指定したあとの括弧｛から｝の間にある記述でその指定先の文字や画像などの見た目を装飾します。 装飾する種類には色や太さや形など様々ありますがその指定をプロパティと呼びます。

値（プロパティの値）

プロパティで指定した内容を値で決定します。色なら赤や青などの色情報、太さならどれくらいの太さかなどを値に指定します。

たとえば下記のような記述では、h1というタグすべての文字色を赤にしてくださいという指示に当たります。

h1 { color:red }
