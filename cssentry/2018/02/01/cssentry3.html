よく使うCSSプロパティ

現在ではCSSは様々なプロパティを設定できます。これらも前の章と同様すべてを暗記する必要は一切ありません。基本的にはほかのWebサイトで見かけた表現は画像や動画、Flashのような他のプログラム以外のものであればすべてCSSで表現可能です。今回はその中でも頻度の高いプロパティを見ていきます。

その中でもフォント系のプロパティはとてもよく利用するのでその中でもよく使うものを見ていきましょう。

フォント色

colorというプロパティでフォントの色を変えることができます。aタグやspanタグでよく利用します。書き方は下記のように記述します。

span.akamoji{
  color:#ff0000;
}

例で出しているの#ff0000はRGB値といいプロパティ値に設定できる種類がいくつかあります。その種類は下記の4種類で実際の色などはサイトなどにまとまっているので参考に探してみてください。

  • RGB値-#000000のよう16進数6桁で表現
  • RGB値短縮-#000のように16進数3桁で表現
  • RGB10進数-rgb(0,0,0)のようにRGB値を直接指定する表現
  • 変数-redのようにあらかじめ指定されている色を指定

具体的には下記のように記述するとフォントの色が変わって表示されます。

[style.css]

.aka{
    color:#ff0000;
}

.blue{
    color:#00f;
}

.yellow{
    color:rgb(255,255,0);
}

.green{
    color:green;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<p class="aka">赤色</p>

		<p class="blue">青色</p>

		<p class="yellow">黄色</p>

		<p class="green">緑色</p>

	</body>
</html>

フォントサイズ

font-sizeというプロパティで文字の大きさを変更できます。これもaタグやspanタグなどで多用します。color同様プロパティ値に設定できる単位の種類は多数ありその中でもよく使うものを書きに記述します。

基準となる文字サイズのパーセンテージ計算でサイズを算出

  • %指定
    基準となる文字サイズを１としてサイズを算出する。2emは基準となる文字２つ分の大きさとなります。
  • em指定
    基準となる文字サイズの小文字の英語xを１としてサイズを算出する。1exは基準の小文字xの大きさと同じになります。
  • ピクセル指定
    ピクセルでサイズを指定します。一番利用しやすい指定方法です。
  • 変数指定
    smallやlargeなどあらかじめ決められた変数の大きさを指定する方法です。
ここでいわれる基準となる文字サイズはhtmlタグや親要素でfont-sizeが指定されていればそのサイズになります。指定されていない場合はブラウザで違いますが16px（ピクセル）が一般的のようです。

具体的な利用例は下記のようになります。
[style.css]

.percent{
    font-size:100%;
}

.doublepercent{
    font-size:200%;
}

.emsize{
    font-size:1em;
}

.doubleemsize{
    font-size:2em;
}

.exsize{
    font-size:1ex;
}

.doubleexsize{
    font-size:2ex;
}

.pixel16{
    font-size:16px;
}

.pixel24{
    font-size:24px;
}

.small{
    font-size:small;
}

.x-large{
    font-size:x-large;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<p class="kijun">基準のサイズ</p>
		<p class="kijun">基準の英語小文字x</p>


		<p class="percent">100%を指定</p>

		<p class="doublepercent">200%を指定</p>


		<p class="emsize">em指定</p>
		<p class="doubleemsize">2em指定</p>

		<p class="exsize">ex指定</p>
		<p class="doubleexsize">2ex指定</p>


		<p class="pixel16">16ピクセルを指定</p>
		<p class="pixel24">24ピクセルを指定</p>

		<p class="small">smallを指定</p>
		<p class="x-large">x-largeを指定</p>

	</body>
</html>

フォントの種類

font-familyというプロパティで表示するフォントの種類を決めることができます。フォントは基本的にはそれぞれ個人の主観なのでどれが一般的なフォントだとかはありません。多くの人が見やすいフォントなどはあるとおもいますが結局人によっては変わってしまうため議論が尽きません。

そしてフォントはクライアントにそのフォントがインストールされていない場合を考え複数指定することができます。左から優先順位がつけられそのフォントがなければ次のフォント、それもなければその次といった感じで採用されます。下記のようにしていします。

body{
    font-family:'Hiragino Kaku Gothic ProN', 'ヒラギノ角ゴ ProN W3', Arial;
}

普通は上記のように記述しますが下記ではフォントが変わるのを見るために段落それぞれにフォントを指定してみます。 [style.css]

.gothic{
    font-family:"MS Gothic";
}

.serif{
    font-family:"sans-serif";
}

.multi{
    font-family:'Hiragino Kaku Gothic ProN', 'ヒラギノ角ゴ ProN W3', Arial;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<p class="kijun">基準となるフォント</p>

		<p class="gothic">ゴシックで記述した文字</p>

		<p class="serif">セリフで記述した文字</p>

		<p class="multi">複数指定してあるがヒラギノがないのでArial</p>

	</body>
</html>

行間

文章の文字の行間を指定することができます。line-heightというプロパティでできますがこれもフォントサイズと同じ単位で指定することができます。

  • %指定
    基準となる文字サイズのパーセンテージ計算でサイズを算出
  • em指定
    基準となる文字サイズを１としてサイズを算出する。2emは基準となる文字２つ分の大きさとなります。
  • ex指定
    基準となる文字サイズの小文字の英語xを１としてサイズを算出する。1exは基準の小文字xの大きさと同じになります。
  • ピクセル指定
    ピクセルでサイズを指定します。
  • 変数指定
    smallやlargeなどあらかじめ決められた変数の大きさを指定する方法です。

文字サイズとほぼ同じ考えで記述するのが良いです。ピクセルなどで指定すると文字サイズなどを変更した際に融通が利かなくなる場合が多々あります。

[style.css]

.danraku150{
    line-height:150%;
}

.danraku200{
    line-height:200%;
}

.colorred{
    color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<p class="danraku150">
			文章の行間の大きさを指定することができます。<br>
			line-heightに150%を指定しているので行全体の高さが150%になります。<br>
			上下に割り振られるので上25%下25%の余白ができます。<br>
		</p>
		<p class="colorred">ここから行間の大きさがかわります</p>
		<p class="danraku200">
			文章の行間の大きさを指定することができます。<br>
			line-heightに200%を指定しているので行全体の高さが200%になります。<br>
			上下に割り振られるので上50%下50%の余白ができます。<br>
			結果的に上下の行間50%が二つ連なるので一文字まるまるあくことになります。
		</p>

	</body>
</html>

テキスト装飾

文章に下線を引いたり打ち消し線を引くことができます。text-decorationというプロパティで装飾することができます。このプロパティに設定できるプロパティ値は下記の5つです。

  • none
    テキストに何の装飾もされません。aタグのように初期値がnone以外の場合にcssで装飾を消したいときに使います。
  • underline
    テキストに下線が引かれます。
  • overline
    テキストの上に線が引かれます。あまり利用しません。
  • line-through
    テキストに打ち消し線（真ん中に線）が引かれます。
  • blink
    テキストが点滅します。現在はあまり見られません。（少しダサいです）

[style.css]

.none{
    text-decoration:none;
}

.underline{
    text-decoration:underline;
}

.overline{
    text-decoration:overline;
}

.line-through{
    text-decoration:line-through;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<p class="none">テキストに何も装飾されません</p>

		<p class="underline">テキストに下線が引かれます</p>

		<p class="overline">テキストの上に線が引かれます</p>

		<p class="line-through">テキストに打ち消し線が引かれます</p>

	</body>
</html>
