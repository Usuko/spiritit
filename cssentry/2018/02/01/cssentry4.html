access_time 2018年02月01日

よく使うボックス系プロパティ

現在では様々なWebサイトがあります。タグで表現された要素は基本的には四角い形をしてパネルのように敷き詰められてWebサイトがデザインされています。その四角い形の要素をCSSでデザインできるのでそのプロパティを見ていきましょう。

サイズ

ボックスのサイズを指定してみましょう。サイズはwidthというプロパティとheightというプロパティで設定可能です。

  • width:横幅を指定
  • height:縦幅を指定
    このプロパティに設定するプロパティ値は%やpxで指定して行きます。今回はサイズに関しては基本的なことしか書きませんが実はいろいろ細かい設定や知識が必要です。
  • %指定
    %で指定すると基本的にはブラウザ横幅に対するパーセンテージで表示されます。 ※縦幅の%指定は横幅とは少し違います。縦幅を指定した要素の子要素の場合にのみ%指定が算出されます。
  • px指定
    ピクセルという単位で大きさが指定できます。基本的には画面解像度と考えてよいです。その単位で指定をすることができます。

下記のような記述で指定できます。・・・・がこの段階ではボックスは色も境目も書かれていないためどのような形かわかりません。次の色指定と合わせて確認していきましょう。

.box1{
	width:50%;
	height:300px;
}

背景色

背景色を変えてみましょう。背景色はbackground-colorというプロパティで装飾することができます。プロパティ値の種類はfontで見た文字色とおなじで下記の４種類です。

  • RGB値-#000000のよう16進数6桁で表現
  • RGB値短縮-#000のように16進数3桁で表現
  • RGB10進数-rgb(0,0,0)のようにRGB値を直接指定する表現
  • 変数-redのようにあらかじめ指定されている色を指定

[style.css]

.aka{
    background-color:#ff0000;
}

.blue{
    background-color:#00f;
}

.yellow{
    background-color:rgb(255,255,0);
}

.green{
    background-color:green;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<div class="aka"></div>

		<div class="blue"></div>

		<div class="yellow"></div>

		<div class="green"></div>

	</body>
</html>

ボーダー（枠線）

枠線を引いてみましょう。borderというプロパティで枠線を記述できます。しかしながらborderというプロパティは様々な派生のプロパティとプロパティ値の指定があります。まずは四方すべての枠線を記述する書き方ですが

border:線の太さ 線の形 線の色;という指定の仕方があります。 それぞれのプロパティ値に設定する値というものをみていきます。

  • 線の太さ:
    ピクセル指定(1px等) 変数指定(small）などができますが一般的にはピクセル指定が通常です。上記の書き方の場合四方すべての線の太さがこの太さになります。
  • 線の形:
    これはいくつか指定があり下記にまとめます。 すべてを覚える必要はありません
    線なし:
    点線: dotted
    破線: dashed
    実線： solid
    二重線: double
    へこんだ線: groove
    浮き上がった線: ridge
    線の内側がへこむ: inset
    線の内側が浮く<: outset/div>
  • 線の色:
    これは背景色と同様で４種類、例などでは#000000といったrgb値を使います。

つまり、下記のようにstyle.cssとindex.htmlを記述するとみることができます。

[style.css]

.box1{
	width:50%;
	height:150px;
	border:3px solid #ff0000;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<div class="box1">
				テストボックスです
			</div>
		</div>
	</body>
</html>

パディング(padding)

パディングは非常によく使うプロパティです。テキストなどなにがしかのレイアウトをデザインする際にこのパディングを設定しない場合要素の左上に詰まって表示されます。

上記のようにボックス内のテキストが左上に詰まって表示されているように表示されます。この親要素と子要素のあいだのスペースを指定することができます。paddingというプロパティで装飾が可能でプロパティ値にはピクセルで指定するのが一般的です。下記のようにcssとhtmlを記述することで指定ができます。

[style.css]

.box1{
	width:50%;
	height:150px;
	border:1px solid #ff0000;

	padding:10px;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<div class="box1">
				テストボックスです
			</div>
		</div>
	</body>
</html>

マージン

マージンもパディングと同様よく使うプロパティです。パディングでは子要素の間にスペースをつくりました、マージンは他要素との間をしていします。プロパティはmarginというプロパティで設定します。またプロパティ値はパディング同様ピクセルで指定するのが一般的です。

上記のようにボックスレイアウトが二つ連なると基本的には間がありませんが、marginというプロパティを使いスペースを作ることができます。下記のようにcssファイルとhtmlファイルを作成してください。

[style.css]

.box1{
	width:50%;
	height:150px;
	background-color:#ff0000;
}

.box2{
	width:50%;
	height:150px;
	background-color:#ffff00;

	margin:10px;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<div class="wrap">
			<div class="box1">
				テストボックスです
			</div>
			<div class="box2">
				テストボックス２です
			</div>
		</div>
	</body>
</html>
