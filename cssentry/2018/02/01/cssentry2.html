CSS入門[第二回]セレクタ

access_time 2018年02月01日

id属性とclass属性（セレクタの前に）

CSSを記述するうえでhtmlの属性であるid属性とclass属性に触れます。idとclass属性はどちらもhtmlタグを識別するために利用できる属性です。CSSで指定したりJavaScriptなど他のプログラムからわかるようにつける印です。二つの属性はほぼ同じ意味で利用されますが下記の点で役割が違います。

  • id属性は一つのhtmlで同じ名前を使ってはいけない
  • id属性は一つのタグに一つしか指定できない
  • class属性は同じ名前を使ってもよいし一つのタグに複数の名前を指定してもよい

id属性

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<p id="p1">段落１です</p>
		<p id="p2">段落２です</p>
		<p id="p2">段落３です</p>
	</body>
</html>

id属性は上記のようにid=”指定するID”のように指定します。上記ではidの”p1″は重複していませんが“p2″は重複しているのでhtmlとしては間違いです。ただし、現状idが重複しているホームページは山ほどありますしブラウザもid重複でエラーにするものは少ないのが現状です。JavaScriptなどを利用しない場合には問題にはなりませんができるだけ正しいHTMLを記述するようにするため重複してしまわないようにしてください。

実際のプログラム開発では間違った記述でも動作としては正しく動いてしまう場合があります。その場合良かれと思って直したことで違うエラーを引き起こす場合がたくさんあるので直してはいけないことが多いです。一番初めにプログラムを記述した人はとても気を使って記述する必要があります。

class属性

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<p class="p1">段落１です</p>
		<p class="p2">段落２です</p>
		<p class="p1 p2">段落３です</p>
	</body>
</html>

class属性はidと同様でclass=”指定するclass名”のように指定します。id属性とは違い上記の段落３のようにclass=”p1 p2″のように複数同時に指定できます。他のプログラムからは同じclass名を持つものはすべてアクセスできます。

セレクタ

cssでは装飾したい場合に装飾する対象をタグ名やid属性、class属性を利用して指定します。それをセレクタと呼びます。cssの記述のそれぞれの名称は下記のようになります。
セレクタで指定したタグのプロパティの値を指定した値に設定することでそのタグを装飾します。上記はh1タグすべてのcolor(文字色)を赤色に設定しています。 セレクタの種類はいくつかありますが全部を覚える必要はありません。こういう指定の仕方があったなということを覚えておけば必要な時は調べればよいです。下記にセレクタの種類とその記述例をまとめます。

ユニバーサルセレクタ

すべての要素の装飾を行えます。セレクタには記号の「*」アスタリスクを使用します。 基本的には多用しません。

[style.css]

*{
	color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1>ページタイトル</h1>

		<p>サンプル文章</p>

	</body>
</html>

タイプセレクタ

タグ名を指定するセレクタです。利用頻度は最重要です。指定したタグのみ装飾を行うときに使います。例ではh1タグのみを指定します。pタグが装飾されていないことを確認してください。

[style.css]

h1{
	color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1>ページタイトル</h1>

		<p>サンプル文章</p>

	</body>
</html>

[表示結果]

IDセレクタ

id属性を指定するセレクタです。指定したいid名の前に記号の「#」シャープをつけて記述します。その際[タグ名]#[id名]と記述する方法と#[id名]と指定する方法があります。一般的にはタグ名を省略することがおおいです。

[style.css]

#title-1{
	color:red;
}

h1#title-2{
	color:blue;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<h1 id="title-1">ページタイトル</h1>
		<p id="p-1">サンプル文章</p>
		<h1 id="title-2">ページタイトル２</h1>
		<p id="p-2">サンプル文章２</p>
	</body>
</html>

[表示結果]

クラスセレクタ

class属性を利用して指定するセレクタです。指定したいクラスの前に「.」ピリオドをつけて指定します。IDセレクタと同様で[タグ名].[クラス名]のように指定する方法とタグ名を省略して.[クラス名]のように指定する方法があります。利用頻度はこのセレクタが一番高いです。 また、class=”クラス名１ クラス名２”のように半角スペースで区切ることで一つのタグに何個でも指定することができます。

[style.css]
class属性にheadを指定したタグの文字色を赤に futojiを指定したタグの文字サイズを1emに（emは別途説明予定)で且つ太字に pタグの中でclass属性にsampleと指定したタグの文字色を青にするというcssの設定です。

.head{
	color:red;
}

.futoji{
	font-size:1em;
	font-weight:bold;
}

p.sample{
	color:blue;
}

[index.html]
bodyの3つ目の要素「ページタイトル２」ではclassを複数していしているのでその両方の装飾がされていることを確認してください。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<h1 class="head">ページタイトル</h1>
		<p class="futoji">サンプル文章</p>
		<h1 class="head futoji">ページタイトル２</h1>
		<p class="sample">サンプル文章２</p>
	</body>
</html>

[表示結果]

複数セレクタ

複数のセレクタに同じプロパティを設定する場合につかいます。セレクタを記号の「,」で区切ると複数のセレクタに同じプロパティを設定することができます。それぞれのセレクタはIDセレクタやクラスセレクタなど何でも構いません。

[style.css]

h1, .sample, .sample3{
	color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<h1>ページタイトル１</h1>
		<p class="sample">サンプル文章１</p>
		<h2>ページタイトル２</h2>
		<p class="sample2">サンプル文書２</p>
		<h2>ページタイトル３</h2>
		<p class="sample3">サンプル文書３</p>
	</body>
</html>

[表示結果]
セレクタで指定した要素のみが装飾されていることを確認してください。

子孫セレクタ

セレクタを半角スペースで区切って指定すると。一つ目で指定したセレクタ内の要素のうち二つ目で指定したセレクタを指定することができます。この場合一つ目のセレクタの要素を親要素として子要素でも孫要素でも二つ目のセレクタで指定したタグは装飾されます。

[style.css]

.parent  p{
	color:red;
}

[index.html]
子孫セレクタでは親要素である<div class=”parent”>内の要素のすべてのpタグを装飾できます。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1 class="head">ページタイトル</h1>
		<p class="futoji">サンプル文章</p>

		<div class="parent">

			<h1 class="head futoji">ページタイトル２</h1>
			<p class="sample">サンプル文章２</p>

			<div class="child">

				<p class="sample2">サンプル文章３</p>

			</div>

		</div>
	</body>
</html>

[表示結果]
サンプル文書２もサンプル文書３という文字双方の文字色が赤くなっていることを確認してください。

子セレクタ

子孫セレクタに似ていますが二つのセレクタを記号のだいなり「>」で区切ります。この場合子孫セレクタと違い一つ目のセレクタで指定した親からみて子の要素のみ装飾されます。

[style.css]

.parent > p{
	color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1 class="head">ページタイトル</h1>
		<p class="futoji">サンプル文章</p>

		<div class="parent">

			<h1 class="head futoji">ページタイトル２</h1>
			<p class="sample">サンプル文章２</p>

			<div class="child">

				<p class="sample2">サンプル文章３</p>

			</div>

		</div>
	</body>
</html>

[表示結果]
子孫セレクタと異なり孫要素であるサンプル文章３の文字色が赤くなっていないことを確認してください。

隣接セレクタ

子セレクタに近いセレクタで隣接セレクタというのがあります。このセレクタでは二つのセレクタを記号の「+」で区切ります。この場合一つ目のセレクタと同じ階層の次の要素が二つ目のセレクタで指定した要素の場合に装飾します。

[style.css]
一つ目のセレクタでクラス属性を指定しているのでclass属性が”parent”を含む要素で同列の次の要素がh1タグの場合それを装飾します。

.parent + h1{
	color:red;
}

[index.html]
class=”parent”の子要素ではなく同列の要素である「ページタイトル３」が装飾されていることを確認してください。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1 class="head">ページタイトル</h1>
		<p class="futoji">サンプル文章</p>

		<div class="parent">

			<h1 class="head futoji">ページタイトル２</h1>
			<p class="sample">サンプル文章２</p>

			<div class="child">

				<p class="sample2">サンプル文章３</p>

			</div>

		</div>
		<h1>ページタイトル３</h1>
	</body>
</html>

疑似クラス

セレクタのうちで少しちょっと動作が違うのですが、リンクを表示する場合に訪問済みのリンクとまだ見たことないリンクで色が違うことがあるとおもいます。そのようなタグの状態によって色を変えたりすることができるセレクタです。 疑似クラスではセレクタ：疑似クラスのように記述します。セレクタの状態が疑似クラスの状態であるときにそのプロパティが適用される仕組みです。

未訪問（link）

a:link{
    color:red;
}

訪問済み（visited）

a:visited{
    color:red;
}

マウスカーソルが上に乗っている（hover）

a:hover{
    color:red;
}

フォーカスされている（focus）

input:focus{
   background:gray;
}
フォーカスとはテキストボックスなどを入力するときのように選択されている状態のことを指します

疑似要素

疑似クラスのように指定した要素の一つ目の要素や前後にスタイルを挿入したり疑似的に要素を追加するようなセレクタです

一つ目の子要素（first-child）

何かの子要素のうち一つ目のpタグを装飾する場合下記のように記述します。基本的には子孫セレクタと組み合わせて使われることがおおいです。

[style.css]

p:first-child{
    color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1>ページタイトル１</h1>
		<div>
			<p class="sample3">サンプル文書１</p>
		</div>

	</body>
</html>

最初の要素の一行目（first-line)

子要素のうち一行目を修飾する場合に記述します。
[style.css]

p:first-line{
    color:red;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1>ページタイトル１</h1>
		<div>
			<p class="sample3">
				１行目のみ色が明るくなり
				<br>２行目以降は装飾がはずれて色がもとにもどります。
			</p>
		</div>

	</body>
</html>

最初の要素の一文字目（first-letter）

最初の要素の一文字目を装飾したい場合に記述します。
[style.css]

p:first-letter{
    color:red;
}

[index.html]

	</head>
	<body>

		<h1>ページタイトル１</h1>

			<p class="sample3">

要素の直前直後（beforとafter）

要素の直前や直後にスタイルを挿入したい場合に使うセレクタです
[style.css]

.comment:before{
	content:"「";
	color:red;
}

.comment:after{
	content:"」";
	color:blue;
}

[index.html]

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>

		<h1>ページタイトル１</h1>
		<div>
			<span class="comment">おはよう</span>
			などのセリフのような言葉をデザインされた括弧でくくる場合などに利用します。
		</div>

	</body>
</html>

セレクタ一覧

一度上記の説明を読み理解した場合には思い出すのは利用例だけでよいので念のために一覧表にまとめます。

名称 書き方 装飾対象 利用例
ユニバーサルセレクタ *（アスタリスク） すべての要素 *{ color:red; }
タイプセレクタ タグ名 指定したタグすべて h1{ color:red; }
IDセレクタ #id名 指定したidを持つ要素 #title{ color:red; }
クラスセレクタ .class名 指定したclassが指定ある要素すべて .head{ color:red; }
複数セレクタ セレクタ,セレクタ,セレクタ 指定したセレクタの要素すべて h1,h2,h3{ color:red; }
子孫セレクタ セレクタ セレクタ 一つ目のセレクタの子孫のうち二つ目のセレクタで指定した要素すべて div p{ color:red; }
子セレクタ セレクタ > セレクタ 一つ目のセレクタの子要素のうち二つ目のセレクタで指定した要素 div > p{ color:red; }
隣接セレクタ セレクタ + セレクタ 一つ目の要素の次の二つ目のセレクタで指定した要素 div + p{ color:red; }
未訪問 セレクタ:link 未訪問の要素 a:link{ color:red; }
訪問済み セレクタ:visited 訪問済みの要素 a:visite{ color:red; }
マウスカーソルが乗っている セレクタ:hover 要素にマウスカーソルが乗った場合 a:hover{ color:red; }
フォーカスされている セレクタ:focus 要素にフォーカスが当たっている場合 input:focus{ background-color:gray; }
一つ目の要素 セレクタ:first-child 指定したセレクタの子要素の一つ目の要素 p:first-child{ color:red; }
要素の一行目 セレクタ:first-line 指定したセレクタの子要素の一行目 p:first-line
要素の一文字目 セレクタ:first-letter 指定したセレクタの子要素の一文字目 p:first-letter{ color:red; }
要素の直前 セレクタ:before 指定したセレクタの直前 span:before{ color:red; }
要素の直後 セレクタ:after 指定したセレクタの直後 span:after{ color:red; }

まとめ

これがすべてではありませんが、cssで装飾する場合に様々な指定方法があることがわかりました。これをすべて暗記する必要はありません。こういうやり方があったということだけでも覚えれば調べたり人に聞いたりすることができるので頭の片隅に残っていればそれで充分です。またよく使うセレクタは結果的に覚えてしまいます。この章では様々なセレクタがあるということを学びました。
