CSS入門[第一回]CSSとはなにか

access_time 2018年02月01日

CSSが作られた理由と特徴

CSSはhtmlの装飾（見た目）に関する記述をするために作られました。 HTMLはコンピュータでも読めるような文章なのでHTML単体では人間に見やすいとは言えません。

現実の本においても、本を最初に書く人は原稿というものがあります。原稿がHTMLに当たります。 本を作る過程においてはその原稿をもとに表紙をつけたり文字に色を付けたり、部分的に文字を太くしたりとした作業を編集者が行います。

この編集者がCSSに当たります。 人間が見やすいようにCSSという別の指示書を使って文章を飾っていきます。

要点は３点です

  • CSSを使いHTMLの見た目を装飾することができる
  • HTMLとCSSが両方あって人が見やすい文章が出来上がります
  • 文章をみるアプリケーションであるブラウザによってCSSの読み方が違う場合がある
htmlが生まれて現在に至るまでにさまざまな企業や団体がブラウザを作りシェアを競う形になりました。そのためブラウザによって使えるタグや使えないタグ、同じように使えるCSS使えないCSSなどブラウザによって動きが違うといった結果を招き、Web製作者やWebプログラマや閲覧者を悩ませ続けてきました、現在でも名残はあります。 ただ、html5とcss3に至った現在ではcssは外部に記述するという点ではそろそろ浸透してきておりhtmlに装飾要素を記述する技術者も減ってきております。cssを学ぶ上で「装飾に関する記述はHTMLから切り離す」ということを前提に学習すると理解が早まるかもしれません。

HTMLとCSSの仕組み

HTMLはタグで記述されています。HTMLのままの文書はページタイトルや本文、目次、画像など文書の構造をタグで区切られて記述されています。CSSはそのHTMLのタグを装飾する記述を書くものです。文字の大きさや色などをプロパティといい、HTMLのどの部分をそのプロパティで修飾するのかということをセレクタというもので指定します。 このHTMLとCSS双方をブラウザが理解し表示しています。

具体的なCSSは下記のように記述されます。セレクタ、プロパティ等の意味は下記の意味です。

  • セレクタ：どのタグを装飾するかを指定するもの
  • プロパティ：セレクタで指定したタグを装飾する種類を指定する（色やサイズ、表示位置など）
  • プロパティ値：プロパティの実際の値（色なら赤や青、サイズなら大文字や小文字など）

CSSを記述する場所

CSSを記述する場所は３種類あります。下記に一つずつ書き方を説明していきます。ここではCSSの細かい書き方を理解する必要はなくCSSをどこに書けばいいかという書き方を理解してください。

HEADタグにSTYLEタグで記述して読み込み

html入門で作成した単純なhtmlを利用します。 下記のようにとstyleタグの要素にCSSを記述することで文書全体にCSSを反映させることができます。

TeraPadを新規に立ち上げてください。 そこに下に記述したコードをコピーして貼り付けます。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>

		<!下記がスタイルシートです>
		<style>
			p{
				color:red;
			}
		</style>
	</head>
	<body>
		<p>初めてHTML文書をかいてみます。</p>
	</body>
</html>

メニューから文字コードを指定して保存します。

文字コードにutf-8を選択してOKボタンを押します

文字コードとは何でしょうか。今現在この文字は日本語になっていますがパソコンの中身には数字（コード）のみが記憶されています。普通のデータもhtmlと同じようにコンピュータが読めるようにコードが保存されていますが人間が見れるように変換しなくてはいけません。その変換に使われるコードが文字コードです。utf-8という文字コードは世界中で標準的に使われているのでhtmlでは一般的にutf-8を使用します。

sample1.htmlとして保存します。

保存したsample1.htmlをダブルクリックしてみましょう。内容が表示されるはずです。 上記コードのstyleの部分がCSSです。pタグのcolor(色を）red（赤）にしますという意味です。 表示結果は指定通りpタグで記述した文字の色が赤くなって表示されます。

外部ファイルに記述して読み込み

まず、外部ファイルを作成します。一般的にはindex.htmlのあるフォルダにcssというフォルダを作成します。下記のようになります。

上記sample1.htmlを保存した場所にcssというフォルダを新規に作成してください。

このcssフォルダの中に先ほどと同じようにstyle.cssというファイルを新規に作成してください。 中身のコードは下記のものをコピーして貼り付けます。

p{
    color:blue;
}

下記のようにファイルが保存されているはずです。

sample1.htmlというフォルダに戻り、これも同じようにsample2.htmlというファイルを新規に作成してください。 内容は下記のコードをコピーして貼り付けます。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
		<link rel="stylesheet" href="./css/style.css">
	</head>
	<body>
		<p>初めてHTML文書をかいてみます。</p>
	</body>
</html>
</pre>

下記のようにsample2.htmlが作成したのでダブルクリックで開いてみます。 先ほどと同じように文字が赤く表示されていればOKです。

上記コードの<link rel=”stylesheet” href=”./css/style.css”>という部分は css（スタイルシート）は”./css/style.css”という場所に書きましたという情報です。 コンピュータはhtmlを読むときにこのstyle.cssというファイルも読み込み表示してくれます。

./css/style.cssは相対パスという記述方法です。「.」の意味はこのファイルを起点にする意味で使います。つまりこのindex.htmlのフォルダから見た/css/style.cssという意味になります。

この記述方法は最も一般的です。HTMLとCSSを完全に切り離して管理できます。そして開発する人やデザイナーもこの方法になれています。

タグの属性に直接スタイルを記述

下記のようにタグに直接style属性という属性を使い記述します。下記ではPタグに直接修飾するcssを記述しています。
※この書き方は非推奨でもあるので見るだけで良いです。

下記のようにタグに直接スタイルを記述します。

<!DOCTYPE html>
<html lang="ja">
	<head>
		<meta charset="UTF-8">
		<title>HTML入門です</title>
	</head>
	<body>

		<p style="color:red">初めてHTML文書をかいてみます。</p>

	</body>
</html>

現在この方法で開発するメリットはなしといってよいです。ためしに記述してみてよかったらきちんと記述するといった形で使われることは有りましたが、エディタも発達しているしSassやjQueryなどの技術もあるためタグに直接スタイルを記述するこの方式は管理面からもやらないほうがよいです。

まとめ

上記のように３種類のcssの読み込み方法を書きましたが最後のタグに直接記述する方法は基本的には使わないでください。現在では非推奨の記述方法（ですがブラウザでは気を聞かせて動作してしまいます）です。冒頭で記述したようにhtmlと切り離すという目的から外れてしまった機能のため現在では非推奨となっております。ただし、プログラム開発やWebページ作成時に一時的に直接タグに書く場合がたまにあるので覚えておいてください。 まとめとして箇条書きにすると

  • 装飾に関する記述はhtmlから切り離しcssで記述する
  • ブラウザによってcssの動きが違う場合がある
  • cssは
  • cssは外部ファイルに記述する方法がある
  • cssはタグに直接記述する方法がある
  • cssでは装飾する対象をセレクタというもので指定する
  • 装飾する種類（色やサイズ）などをプロパティと呼ぶ
  • 装飾するものの実際の指定（色なら赤や青、サイズなら大文字や小文字）などをプロパティ値と呼ぶ
